Scoreboard

This project was for my son. It's a scoreboard running on a Raspberry Pi 3B+ with a 64x32 RGB LED Matrix connected to a RGB Matrix Bonnet built in a Lego case. I found this project on GitHub and thought my son would really like it. It currently lists games/scores for basketball and hockey but can do many more. The Raspberry Pi is updated through Ansible. Cron entries are used to turn the screen off at bedtime and on before school.
